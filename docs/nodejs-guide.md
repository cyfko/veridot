# Node.js Guide (Deprecated)

> [!CAUTION]
> The `dverify` Node.js package is **no longer maintained** and is not compatible with Veridot v3.0.
> This guide is preserved for historical reference only.
>
> For new integrations, use the **Java SDK** (`veridot-core`) which implements Protocol V2
> with TrustAnchor-based security, signed tombstones, and RSA-3072.
>
> See the [Java Guide](java-guide.md) for getting started.
