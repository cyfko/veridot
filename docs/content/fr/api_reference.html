Référence API

Cette page liste les interfaces Java, les constructeurs, les méthodes et les exceptions du projet Veridot.

Interfaces Clés

1. DataSigner

Interface pour signer et émettre des jetons utilisateur.

public interface DataSigner {
    /**
     * Génère un jeton cryptographique signé pour le payload fourni.
     *
     * @param data Le payload à sérialiser et signer.
     * @param configurer Les options de configuration (validité, sequenceId, etc.).
     * @return Le jeton signé.
     * @throws DataSerializationException En cas d'erreur de sérialisation.
     */
    String sign(Object data, Configurer configurer) throws DataSerializationException;
}

2. TokenVerifier

Interface pour valider et extraire localement les payloads des jetons.

public interface TokenVerifier {
    /**
     * Valide la signature, le statut actif (liveness) et la configuration du jeton.
     *
     * @param token Le jeton.
     * @param deserializer Fonction pour désérialiser le payload JSON vers le type T.
     * @return Un conteneur VerifiedData avec les données et le statut.
     * @throws BrokerExtractionException Si les métadonnées ne peuvent être récupérées du broker.
     */
    <T> VerifiedData<T> verify(String token, java.util.function.Function<String, T> deserializer)
        throws BrokerExtractionException;
}

3. TokenRevoker

Interface pour révoquer cryptographiquement des sessions.

public interface TokenRevoker {
    /**
     * Révoque une session spécifique ou toutes les sessions d'un groupe.
     *
     * @param scope Le scope ciblé (ex. Scope.group("user-99")).
     * @param sequenceId L'identifiant unique de la session à révoquer, ou null pour tout le groupe.
     */
    void revoke(Scope scope, String sequenceId);
}

Constructeurs & Méthodes

GenericSignerVerifier

L'implémentation standard des interfaces DataSigner, TokenVerifier et TokenRevoker.

  • public GenericSignerVerifier(Broker broker, TrustRoot trustRoot)
    Initialise une instance configurée en vérification uniquement.
  • public GenericSignerVerifier(Broker broker, TrustRoot trustRoot, String issuerId, java.security.PrivateKey privateKey, byte sigAlg)
    Initialise une instance capable d'émettre (signer) et de valider des jetons.
  • public void close()
    Libère les ressources, arrête les threads d'écoute Kafka ou les pools de connexions SQL.

Exceptions

Exception Classe Parente Description
BrokerExtractionException RuntimeException Levée si la récupération des données du broker échoue ou est désynchronisée.
DataSerializationException Exception Levée si la sérialisation du payload en JSON échoue.
EnvelopeValidationException Exception Levée si une enveloppe binaire ne respecte pas les règles de structure.
RuleValidationException Exception Levée lors du rejet par une règle de validation de claims.
